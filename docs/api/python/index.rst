Continuum Python API
====================

Continuum provides a single runtime-oriented API surface for mixed token/tensor
programs, with backend routing, cache-aware execution, and native extension
acceleration exposed through a compact Python interface.

Use this reference to discover the stable public Python symbols intended for
application and library integrations.
