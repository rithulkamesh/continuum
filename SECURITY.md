# Security Policy

## Supported Versions

Continuum is currently in active development. Security fixes are applied to the latest release on `main`.

## Reporting a Vulnerability

Please report vulnerabilities privately and do not open public GitHub issues.

- Email: <security@rithul.dev>
- Include: affected version/commit, impact, reproduction steps, and any proof-of-concept.

You can expect an acknowledgment within 72 hours. We will keep you informed as we triage, reproduce, and remediate.

## Disclosure

We follow coordinated disclosure and will credit reporters who want public acknowledgment.
