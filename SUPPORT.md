# Support

## Getting Help

- Usage questions and troubleshooting: GitHub Discussions (recommended)
- Bug reports: GitHub Issues
- Security issues: see `SECURITY.md` (private reporting only)

## What to Include

When asking for help, include:

- OS and Python version
- Continuum version (`pip show continuum-ai`)
- Minimal reproducible example
- Relevant logs or stack traces

Clear reproduction details help maintainers respond faster.
